About this Project

Like The Hexenbracken before it, this is another attempt to crowd source a hex crawl. This time I turned the resulting comment thread into a Google Doc, which was then cleaned up with the help of Brendan Strejcek, James Young, Ben Hannigan and Joey Lindsey. A bunch of Python code turns that into the page you see.

A big thank you to Zak Smith for organizing, Ben Hannigan for going above and beyond in editing the Google Doc, and to all the contributors: {{ authors }}.
